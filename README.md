mup'n
=====

The simple FTP client with windows shell extention.

mup'n は Microsoft Windows XP / Windows 7 向けのシェル統合 FTP クライアントです。
エクスプローラーの右クリック・メニューから、選択したファイルの FTP アップロードや
削除をすることができます。

* エクスプローラー上で 2 クリックするだけで、自分の FTP スペースにファイルをアップロードできます。
* アップロードしたファイルを削除したい場合、同じファイルをエクスプローラーで 2 回クリックします。
* ファイルのアップロード完了後、HTTP URL を自動的にクリップボードにコピーしてくれます。
* HTTP URL を短縮 URL に自動的に変換する機能がついています。
* アップロードしたファイルを WWW ブラウザで確認する機能がついています。
